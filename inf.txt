/src
  /components   (Reusable UI components)
  /pages        (Main pages like Dashboard, Equipment, Tasks)
  /store        (Zustand state management)
  /hooks        (Custom hooks for API fetching, logic)
  /utils        (Helper functions)
  /api          (API calls using React Query)
  /assets       (Images, icons)


1. Core Libraries
React (Vite) – Fast, optimized development.
React Router – For handling page navigation.
Zustand – Lightweight state management.
React Query (TanStack Query) – Efficient API fetching and caching.
TailwindCSS + ShadCN/UI – Modern and clean UI components.

Feature	Best Tech to Use
Dashboard & Reports	React Query + Zustand + Recharts
Equipment & Task Management	React Hook Form + Zustand
Calendar View	React Big Calendar / FullCalendar
File Uploads (Photos)	Cloudinary SDK (Frontend Only)
CSV Export	PapaParse (for CSV export)
Filtering & Searching	Zustand + React Table
Notifications & Alerts	React Toastify
